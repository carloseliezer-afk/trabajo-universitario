Sobre el proyecto

Presentacion

Este sitio presenta una propuesta academica para estudiantes que desean iniciar en el desarrollo web. Su objetivo es organizar informacion basica, mostrar recursos utiles y practicar la publicacion de una pagina en una plataforma gratuita.

La pagina fue disenada con HTML, CSS y JavaScript. Incluye navegacion interna, secciones de contenido, un elemento visual, contacto simulado y un pie de pagina, cumpliendo los requisitos de la actividad.

Servicios o contenidos

Temas principales

01

HTML semantico

Estructura basica de una pagina web usando encabezados, secciones, enlaces, imagenes y contenido organizado.

02

Diseno con CSS

Aplicacion de colores, tipografias, distribucion responsive y estilos visuales para mejorar la presentacion.

03

Publicacion web

Uso de plataformas gratuitas como GitHub Pages, Netlify o Vercel para compartir el sitio mediante un enlace publico.

Galeria o recursos

Material de apoyo

Guia rapida

Pasos basicos para crear una pagina: planificar, escribir contenido, disenar, probar y publicar.

Practicas recomendadas

Usar textos claros, enlaces funcionales, contraste adecuado y vista adaptable para celular.

Evidencias

Capturar la pagina local, el panel de publicacion, la vista movil y el enlace publico final.
